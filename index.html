使用git的步骤

1、git config -global user.name '你的用户名'

设置你的用户名（和你github用户名相同）

2、git config -global user.email "email@ XX.com'

设置你的邮箱（和你的github注册邮箱相同）

3、git colne 链接地址

在当前文件夹下下载你的代码仓库

####你编辑了你的代码之后####

4、git add 文件名

提交你的文件到暂存区

5、git commit -m "文件描述"

提交你的代码到本地仓库

6、git push

提交你的代码到远程仓库（github）完成更新代码！这个过程中可能会跳出登录窗口，登录即可
